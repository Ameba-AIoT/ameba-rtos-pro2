Example Description

This example demonstrates how to use a ToF (Time-of-Flight) sensor to display distance measurements in an 8x8 array format.

1. Connect ToF sensor SDA(PE_4) and SCL(PE_3) to AmebaPro2. I2C pin can be modified in vl53l5cx_rtl8735b_i2c.h.
  
2. Run the main function.

3. Get the distance measurements in an 8x8 array format.
